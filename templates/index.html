Bienvenido al Sistema de Alerta Temprana

Este sistema utiliza un enfoque de "Predicción Gradual Adaptativa" para identificar a los estudiantes en riesgo a medida que avanza el ciclo académico.

Utilice las siguientes opciones para entrenar los modelos o realizar nuevas predicciones.

Entrenar un Modelo Realizar una Nueva Predicción

Paso 1: Entrenar Modelo

Suba el archivo histórico consolidado (ej. Data - consolidada.xlsx) para entrenar y re-entrenar los modelos para cada unidad (U1, U2, U3, U4). Cada entrenamiento genera un archivo .pkl más inteligente.

Ir a Entrenamiento

Paso 2: Predecir Estado

Suba un archivo de estudiantes (ej. Para predecir U2.xlsx). El sistema detectará automáticamente qué unidad debe predecir (U1, U2, U3 o U4) basándose en las notas ya registradas.

Ir a Predicción
