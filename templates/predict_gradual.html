Subir Datos para Predicción

Suba aquí el archivo de los estudiantes para quienes desea una predicción. (ej. Para predecir U1.xlsx, Para predecir U2.xlsx).

El sistema detectará automáticamente qué unidad debe predecir basándose en las notas que ya están registradas en el archivo.

¿Cómo funciona?
  • Si sube un archivo sin notas U1...
  • ...el sistema usará modelo_gradual_ud1.pkl para predecir U1.
  • Si sube un archivo con notas U1 pero sin U2...
  • ...el sistema usará modelo_gradual_ud2.pkl para predecir U2.
  • Si sube un archivo con U1, U2 pero sin U3...
  • ...el sistema usará modelo_gradual_ud3.pkl para predecir U3.
  • ...y así sucesivamente.
